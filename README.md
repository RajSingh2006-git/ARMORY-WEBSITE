# ARMORY Agency

**[🚀 View Live Demo](http://ephemeral-gingersnap-8def60.netlify.app)**

A premium, state-of-the-art landing page for an AI Automation and AI Agents agency. This project features a dark, cinematic aesthetic with custom AI-generated visuals, advanced scroll-reveal animations, interactive dashboards, and a sleek modern UI designed to capture high-value enterprise clients.

## 🚀 Features

- **Cinematic Dark Mode Design**: High-contrast, premium aesthetic with subtle glassmorphism and gradient effects.
- **Advanced Scroll Animations**: Custom scroll-reveal sequences built with native IntersectionObserver for smooth entrance effects.
- **Interactive AI Simulation Dashboard**: A mock "Builder" UI featuring dynamic animated network routing paths.
- **Responsive Layouts**: Fully responsive grid architecture tailored for desktop, tablet, and mobile viewing.
- **Custom AI Visuals**: Abstract, futuristic illustrations representing neural networks, embeddings, and cybersecurity.

## 🛠 Tech Stack

- **HTML5**: Semantic and accessible document structure.
- **CSS3 (Vanilla)**: Custom design system using CSS variables, Grid/Flexbox layouts, and keyframe animations.
- **JavaScript (Vanilla)**: Lightweight interactions, scroll observer logic, dynamic statistic counting, and custom SVG path animations.

## 🌐 Deployment

This project is a static website and can be deployed easily to any static hosting provider.

### Quick Deploy (Netlify)
If you have the Netlify CLI installed:
```bash
npx netlify-cli deploy --prod --dir=.
```

### Quick Deploy (Surge)
```bash
npx surge .
```

## 💻 Local Development

No build tools or bundlers are required. Simply clone the repository and open `index.html` in any modern web browser to view the site locally.

```bash
git clone https://github.com/RajSingh2006-git/ARMORY.git
cd ARMORY
open index.html
```
